Análisis de sentimiento

El análisis de sentimiento es el campo de estudio que analiza las opiniones, sentimiento, evaluaciones, actitudes y emociones de las personas a partir del lenguaje escrito. El objetivo principal de este análisis es la creación de un clasificador que permita predecir la polaridad de un texto respecto a un rango específico de posibles valores categóricos.
